# **Water Scarcity Analysis**

Analysis of water scarcity analysis data from TidyTuesday 2023 - Week of 2023-07-18
:::

## Overview

This project explores the **Water Scarcity Analysis** dataset from TidyTuesday, focusing on data visualization and analysis techniques aligned with **SDG 6: Clean Water and Sanitation**.

## Load Required Packages

```{r load-packages}
library(tidyverse)
library(lubridate)
library(here)
library(showtext)
library(ggtext)
library(patchwork)
library(plotly)
```

## Data Import

```{r load-data, eval=FALSE}
# Load data using tidytuesdayR
# library(tidytuesdayR)
# tuesdata <- tt_load('2023-07-18')
# df <- tuesdata$data_name

# Sample data for demonstration
set.seed(123)
df <- data.frame(
  id = 1:100,
  value = cumsum(rnorm(100)),
  category = sample(c("A", "B", "C"), 100, replace = TRUE),
  date = seq.Date(from = as.Date("2023-07-18"), by = "day", length.out = 100)
)

glimpse(df)
```

## Visualizations

```{r visualization, eval=FALSE}
ggplot(df, aes(x = date, y = value, color = category)) +
  geom_line(size = 1.2, alpha = 0.7) +
  labs(title = "Water Scarcity Analysis") +
  theme_minimal()
```
