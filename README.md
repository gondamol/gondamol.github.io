# Nic | Data Portfolio

Public source for the Quarto website at
`https://gondamol.github.io`

## What this repository contains

1. Portfolio pages and site content in `.qmd` files
2. Site configuration in `_quarto.yml`
3. Styling in `custom.scss` and `styles.scss`
4. Generated site output in `_site`

## Main content areas

1. `index.qmd` for home
2. `about/` for profile and background
3. `projects/` for portfolio work
4. `research/` for thesis and research pages
5. `cv/` for curriculum vitae
6. `blogs/` and `posts/` for blog index and posts
7. `personal/` for life page
8. `resources/` for learning resources

## Local development

Prerequisites

1. Quarto
2. R only if you plan to run R code chunks

Run locally

```bash
quarto preview
```

Build static output

```bash
quarto render
```

## Deployment

This site is hosted on GitHub Pages.

Typical publish flow

1. Update content
2. Run `quarto render`
3. Commit and push to GitHub
