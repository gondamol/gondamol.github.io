5. Advanced Analytics Projects

Project 1: Sentiment Analysis - Kenya Twitter Health Discourse ⭐

Data Source: Twitter API / Kaggle Kenya tweets dataset

Analyses: - Topic modeling: What health issues trend? - Sentiment analysis: Public opinion on NHIF, SHA - Network analysis: Influential voices in health policy - Temporal patterns: During COVID vs now - Geographic patterns: Urban vs rural sentiment

Methods: - NLP with tidytext - BERT for sentiment classification - LDA for topic modeling - Network graphs with igraph

Visualizations: - Word clouds (positive/negative) - Sentiment time series - Topic evolution over time - Network graph of health discourse - Interactive topic explorer

Project 2: Machine Learning - Health Insurance Enrollment Prediction ⭐

Data Source: Kaggle health insurance dataset + simulated Kenya data

Problem: Predict which informal sector workers will enroll in NHIF

Methods: - Logistic regression (baseline) - Random forest - XGBoost - Neural network - Model comparison and selection

Features: - Demographics - Employment type - Income level - Distance to health facility - Previous health shocks - Behavioral nudges received

Visualizations: - ROC curves comparing models - Feature importance plots - Confusion matrices - SHAP values for interpretability - Interactive prediction calculator

Output: Policy recommendations for improving enrollment

Project 3: Predictive Analytics - Maternal Health Risk Scoring ⭐

Data Source: Kaggle maternal health dataset

Problem: Early warning system for maternal complications

Methods: - Risk score development - Time-to-event analysis - Classification models - Threshold optimization

Features: - Age, parity, BMI - Blood pressure, blood sugar - Antenatal care visits - Distance to facility - Socioeconomic factors

Visualizations: - Risk score distribution - Calibration curves - Decision curve analysis - Interactive risk calculator

Application: Mobile health (mHealth) integration for community health workers

Project 4: Time Series Forecasting - Disease Outbreak Prediction

Data Source: Kenya Ministry of Health disease surveillance data / WHO data

Focus: Malaria, cholera, dengue patterns

Methods: - ARIMA models - Prophet (Facebook) - LSTM neural networks - Seasonal decomposition

Visualizations: - Forecast with confidence intervals - Seasonal patterns - Outbreak probability map - Early warning dashboard
